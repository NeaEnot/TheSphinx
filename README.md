# TheSphinx

Password manager with the possibility of semi-automatic synchronization of storage with the cloud.

Two passwords are used for encryption - the password from the storage and the password from the encrypted fields.

When the application is first launched, it will ask you to enter the password for the storage, and then create a storage.dat file in which all data will be stored.

You can download the application from the [link](https://disk.yandex.ru/d/673jNfoSUraw5g).
